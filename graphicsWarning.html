Graphics Warning

A warning is shown when painted graphics might not be pixel-perfect.

Web Replay works by recording and replaying the browser's interactions with the underlying operating system. When painting graphics, this includes calls to render text, widgets (checkboxes, etc.), and so forth. When paused at a JS statement, Web Replay will paint graphics reflecting the DOM/CSS state at that point. This painting might require calling into the system in different ways from what happened while originally recording. In such cases Web Replay will attempt to reconstruct the exact graphics which would have been seen at that point, but the result might not be 100% accurate.

Web Replay's handling of these cases will improve over time. See these issues #50 and #116 for more information.
